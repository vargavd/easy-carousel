Easy Carousel plugin

Before the examples:

  • This is a img slideshow plugin for jQuery, with builtin lightbox capability.
  • For more information and the parameters, see the github repo.
  • You have to include jQuery, preferably in the head section:
<head>
   <script src="https://code.jquery.com/jquery-1.12.4.min.js"<</script>
</head>
  • The html code for the images is similar in every example:
<div id="easy-carousel-1">
   <img src="imgs/2/1.jpg" alt="Lorem ipsum dolor sit amet, consectetur adipiscing elit" />
   <img src="imgs/2/2.jpg" alt="Lorem ipsum dolor sit amet, consectetur adipiscing elit" />
   <img src="imgs/2/3.jpg" alt="Lorem ipsum dolor sit amet, consectetur adipiscing elit" />
   <img src="imgs/2/4.jpg" alt="Lorem ipsum dolor sit amet, consectetur adipiscing elit" />
   <img src="imgs/2/5.jpg" alt="Lorem ipsum dolor sit amet, consectetur adipiscing elit" />
   <img src="imgs/2/6.jpg" alt="Lorem ipsum dolor sit amet, consectetur adipiscing elit" />
</div>

Example 1:

  • default parameters
  • images with same size
<script>
   $('#easy-carousel-1').easyCarousel();
</script>

Example 2:

  • custom image sizes and visible image number
  • images with different sizes
<script>
   $('#easy-carousel-2').easyCarousel({
      imgWidth: '350px',
      imgMaxHeight: '200px',
      visibleImgCount: 4
   });
</script>

Example 3:

  • custom style
  • images with different sizes

   $('#easy-carousel-3').easyCarousel({
